== Description ==

List building, create, send and track e-mails for Contact Form 7

**[Plugin Demo](http://azexo.com/automation/)**

This is bridge plugin between **[Marketing Automation by AZEXO](https://wordpress.org/plugins/marketing-automation-by-azexo/)** and [Contact Form 7](https://ru.wordpress.org/plugins/contact-form-7/).

= Main Features =

* Collecting of form submissions with [Admin Columns](https://wordpress.org/plugins/codepress-admin-columns/) and [Advanced Custom Fields](https://wordpress.org/plugins/advanced-custom-fields/) support
* Responsive email drag & drop composer - with **StampReady** email-template format support
* SMTP-Ready - compatible with Postman, WP Mail SMTP, Easy WP SMTP, Easy SMTP Mail, WP Mail Bank plugins
* Site visitors segmentation
* Polls, Surveys
* SMS/Email notifications
* SMS/Email autoresponder
* Double opt-in Phone/Email verification
* SMS/Email newsletters
* Email bounce counting in **[PRO version](https://codecanyon.net/item/marketing-automation-by-azexo/21402648)**
* Visitor country and city detection
* Simple CRM with [AZEXO Analytics](https://wordpress.org/plugins/azexo-advertising-analytics/), [Admin Columns](https://wordpress.org/plugins/codepress-admin-columns/) and [Advanced Custom Fields](https://wordpress.org/plugins/advanced-custom-fields/)
* Supported all features of **[Marketing Automation by AZEXO](https://wordpress.org/plugins/marketing-automation-by-azexo/)**


== Installation ==

1. Upload the `azh_cf7_newsletter` folder to your plugins directory (e.g. `/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the instructions
